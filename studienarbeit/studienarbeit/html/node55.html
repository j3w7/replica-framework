Overview

This sub section offers an overview of all Replica Framework modules in table .

Table: Overview of all Replica Framework plugins
Bundle Postfix,
Module Name
Description Dependencies
owlapi
The OWLAPI lib Plug-in plugin contains the patched version of the OWLAPI library. -
core
The core plugin contains the Shared Ontology Implementation and Shared Ontology builder which is required when applying OWLAPI updates. owlapi
app
The application plugin contains interfaces and classes required to build applications like server or client instances. core
comm
The communication module contains all communication related components of the Replica Framework. Especially the CommManager is included in this plugin. core
dictionary
The dictionary plugin plugin holds interfaces and classes related to the Dictionary concept of the Replica Framework. core
fragments
The fragments plugin bundle incorporates the Shared Ontology Fragment concept related components of the Replica Framework. core
changes
The Change Management plugin holds special implementations of the OWLReplicaOntology interface that actualize all necessary parts of the Change Management concept of the framework. core, fragments
neonplugin
The NeOn Toolkit plugin includes the implementation of the NeOn Toolkit plugin. core, comm
plicies
This plugin contains components required for policy management. It has not yet been implemented at the time of writing. core
query
This plugin holds Query Management related parts. It has not yet been implemented at the time of writing. core
demo
This plugin contains an implementation of a standalone demonstrator application that presents features of the Replica Framework. core, comm
