Core module

The core module is the one on which all other modules depend on. It contains an implementation of the Shared Ontology described in section and the class OWLReplicaManager which is the entry point to the framework. OWLReplicaManager has the same functionality as the OWLManager in the OWLAPI.

Moreover the core module contains a builder which generates an abstract shared ontology implementation. This builder can be used from the command line and was made for the purpose of reducing maintenance costs in case the OWLAPI is updated.
